1. sudo apt install git<br/>

3. sudo apt install go<br/>
4. git clone https://github.com/GivenBY/Igscrapper <br/>
5. cd Igscrapper<br/>
6. python3 Igscrapper.py -u \<username to scan\> <br/>
Or</br>
6. go run main.go
